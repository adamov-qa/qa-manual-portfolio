# QA Manual Portfolio — SauceDemo

## Project under test
- Application: SauceDemo (web e-commerce demo)
- URL: https://www.saucedemo.com/
- Type: Web application

## Goal
Build a QA portfolio with real manual testing artifacts: test plan, test cases, checklists, bug reports, and test summary.

## Scope (initial)
Core user flow:
- Login
- Products page
- Add/Remove from cart
- Cart page
- Checkout

## Repository structure
- `test-plan/` — short test plan
- `test-cases/` — test cases (positive & negative)
- `checklists/` — smoke & regression checklists
- `bug-reports/` — bug reports with severity/priority
- `test-summary/` — test execution summary
